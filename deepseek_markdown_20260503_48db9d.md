# 🍽️ Telegram Web App — Ресторан с системой лояльности по ID

## 🔥 Фишки
- **Идентификация по Telegram ID** — без регистрации
- **Персональные бонусы** за бронирования и лайки (+50 за стол, +5 за меню)
- **Уровни лояльности**: 3+ брони → постоянный гость (кешбэк 10%)
- **История операций** каждого пользователя
- **Промокоды** с копированием
